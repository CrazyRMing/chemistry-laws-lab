# 專案開發筆記 (PROJECT_NOTES)

## 開發背景與目標
為了配合化學教學，建立一個直觀、好懂且視覺效果精美的互動網頁。
- **定比定律實驗室**：以「氫（H）與氧（O）合成水（$H_2O$）」為例。不論使用者輸入多少原子，合成的水分子中，氫與氧的個數比恆為 $2:1$，質量比恆為 $1:8$。多餘的原子將會以游離狀態殘留。
- **倍比定律實驗室**：以「碳（C）與氧（O）的反應」為例。
  - 當氧氣不足時合成一氧化碳（$CO$，個數比 $1:1$）。
  - 當氧氣充足時合成二氧化碳（$CO_2$，個數比 $1:2$）。
  - 進而驗證「與固定質量碳結合的氧，其質量呈簡單整數比 $1:2$」。
