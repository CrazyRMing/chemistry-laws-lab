化學基本定律虛擬實驗室 (Chemical Laws Virtual Lab)

一個基於 HTML5 Canvas 與 Vanilla JS 實作的互動網頁，旨在幫助學生視覺化理解兩個化學基本定律：

  1. 定比定律 (Law of Definite Proportions)
  2. 倍比定律 (Law of Multiple Proportions)

專案特色

  • 零外部依賴：完全採用純原生 HTML5、CSS3 與 ES6+ JavaScript，下載後按兩下即可直接在任何瀏覽器中離線開啟。
  • 現代視覺美學：採用高對比度的深色模式與半透明毛玻璃（Glassmorphism）卡片，搭配精緻的按鈕動畫與流暢的物理碰撞模擬。
  • 即時數據回饋：反應前後的原子數量、分子個數、游離原子、以及質量的比例變化皆會即時計算並呈現在漂亮的表格中。

運行方式

  1. 將本專案下載/複製到您的電腦中。
  2. 連按兩下開啟 index.html 即可在瀏覽器中運行。

實驗說明

1. 定比定律（氫氧合成水）

  • 反應方程式：2H₂ + O₂ ➔ 2H₂O
  • 質量關係：氫原子量為 1，氧原子量為 16。水分子中，氫與氧的質量比恆為 2 : 16 = 1 : 8。
  • 實驗方式：拖曳滑桿設定初始的氫原子（H）與氧原子（O）個數，點擊「開始反應」。你將看到原子隨機運動碰撞並結合成水分子，多餘的原子則保持游離狀態。數據面板會即時計算生成的質量比是否符合 1 : 8。

2. 倍比定律（碳氧反應）

  • 反應方程式：
    • 一氧化碳：C + O ➔ CO
    • 二氧化碳：C + 2O ➔ CO₂
  • 質量關係：碳原子量為 12，氧原子量為 16。
    • 在 CO 中，12 克碳與 16 克氧結合。
    • 在 CO₂ 中，12 克碳與 32 克氧結合。
    • 相同質量的碳所結合的氧質量比為 16 : 32 = 1 : 2（簡單整數比）。
  • 實驗方式：點擊「一氧化碳模式」或「二氧化碳模式」，設定原子個數並開始反應。數據面板會對比兩種模式下，與相同碳原子結合的氧原子的倍數關係。

專案開發筆記 (PROJECT_NOTES)

開發背景與目標

為了配合化學教學，建立一個直觀、好懂且視覺效果精美的互動網頁。

  • 定比定律實驗室：以「氫（H）與氧（O）合成水（H₂O）」為例。不論使用者輸入多少原子，合成的水分子中，氫與氧的個數比恆為 2:1，質量比恆為 1:8。多餘的原子將會以游離狀態殘留。
  • 倍比定律實驗室：以「碳（C）與氧（O）的反應」為例。
    • 當氧氣不足時合成一氧化碳（CO，個數比 1:1）。
    • 當氧氣充足時合成二氧化碳（CO₂，個數比 1:2）。
    • 進而驗證「與固定質量碳結合的氧，其質量呈簡單整數比 1:2」。

踩坑與心得

💡 Windows 虛擬環境路徑字元集問題：
在 Windows 中如果專案路徑包含中文（如「教育相關/因材網製作/數據圖表繪製」），使用 Python 3.12 建立 venv 安裝 manimgl 時，因為 setuptools >= 82.0.0 移除了 pkg_resources 容易導致載入失敗。解決方法是手動將 setuptools 降級至 81.0.0 以下（如 80.10.2），便能成功解決 ModuleNotFoundError。

AntiGravity 專案協同開發指南

本專案是用於說明定比定律 (Law of Definite Proportions) 與 **倍比定律 (Law of Multiple Proportions)** 的互動網頁動畫實驗室。

開發技術棧

  • 核心：HTML5 + Vanilla JavaScript (ES6+)
  • 樣式：Vanilla CSS (深色模式配色的玻璃擬物化 Glassmorphism 設計，配合 Outfit/Inter 字型)
  • 動畫技術：HTML5 Canvas 粒子引擎

協作規則

  1. 極簡與效率 (Ponytail 模式)：
    • 儘可能避免引入外部大型 JS/CSS 庫。
    • 保留程式碼的簡單與純粹性，無須編譯即可透過瀏覽器直接開啟（方便於因材網系統、外掛中離線使用）。
  2. 視覺效果與美學 (Aesthetics)：
    • 使用精美的高對比度漸層與毛玻璃卡片。
    • 粒子碰撞結合必須有流暢的動態過渡動畫與反彈效果。
  3. 專案筆記更新：
    • 每當階段性功能完成或有新發現，必須隨時更新 PROJECT_NOTES.md，並在收工前編譯/同步至 專案報告.html。
