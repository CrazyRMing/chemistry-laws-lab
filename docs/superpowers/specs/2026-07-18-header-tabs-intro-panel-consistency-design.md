# 頁首、頁籤與首頁介紹面板一致性設計

日期：2026-07-18

## 目標

統一四個正式教學頁面的頁首識別與頁籤狀態，並讓定比定律與倍比定律首頁的第一步介紹面板採用相同視覺語言。

涵蓋頁面：

- `index.html`
- `quiz.html`
- `multiple_proportions.html`
- `multiple_proportions_explain.html`

## 核准的設計

### 1. 例題頁左上角識別

- `quiz.html` 與 `multiple_proportions_explain.html` 左上角頁碼方塊都顯示「挑戰」。
- 兩個定律首頁 `index.html` 與 `multiple_proportions.html` 維持顯示「01」。
- 定比例題不再依題目進度把左上角方塊更新成 `09` 等數字；題目進度仍由頁面既有的題次文字呈現。

### 2. 首頁第一步介紹面板

- 以倍比定律首頁第一步的手繪框風格為共同基準。
- 定比定律首頁第一步加入相同的 Canvas 內層手繪框、置中標題與置中說明文字配置。
- 定比定律保留自己的標題與說明內容，不複製倍比定律的教學文案。
- 外層面板尺寸、響應式縮放與 Canvas DPR 處理沿用既有機制。
- 後續步驟的教學圖、動畫及圖表不因本次一致性修改而改變。

### 3. 頂部頁籤狀態

- 當前頁籤：白底、普魯士藍文字與普魯士藍框線。
- 非當前頁籤：普魯士藍底、白字與普魯士藍框線。
- 任一頁面只允許一個頁籤具有 `active` 狀態。
- 滑入非當前頁籤時使用較亮的普魯士藍變體，不使用白底，以免與當前頁混淆。
- 手機維持既有 2×2 排列，狀態顏色規則與桌面一致。

## 實作邊界

- 優先修改既有 HTML、CSS 與對應 Canvas 繪製函式，不建立新元件框架或新增相依套件。
- 共用頁籤狀態只由 `style.css` 的 `.soil-tab-btn` 與 `.soil-tab-btn.active` 管理。
- 例題頁識別文字由 HTML 提供；若既有 JavaScript 會更新定比例題左上角內容，應移除該項更新，但不改變題目導航狀態。
- 首頁介紹面板只調整第一步繪圖分支，不修改其他步驟。
- 不改動圖表、資料點、圖例、化學元素及教學圖內部既有配色。
- 四頁共用樣式或腳本若有修改，需同步更新資源版本參數，避免手機載入舊快取。

## 驗證方式

### 自動檢查

- 確認兩個例題頁左上角均顯示「挑戰」，且定比例題腳本不再覆寫為數字。
- 確認每個正式頁面恰有一個 `.soil-tab-btn.active`。
- 確認 inactive 頁籤為普魯士藍底白字，active 頁籤為白底普魯士藍字。
- 確認手機頁籤仍採兩欄排列且沒有橫向溢位。
- 確認定比定律第一步包含與倍比定律一致的內層手繪框繪製結構。
- 執行完整 Node 測試與四個主要 JavaScript 語法檢查。

### 視覺檢查

- 桌面：四頁都只能看到一個白色當前頁籤，其餘三個為普魯士藍。
- 390px 手機：頁籤維持 2×2，文字不超框，當前頁清楚可辨。
- 比較兩個首頁第一步：外框、內層手繪框、標題與說明位置一致，各自文案正確。
- 比較兩個例題頁：左上角皆為「挑戰」，題目進度資訊仍可見。

## 非目標

- 不修改教學內容、題目數量、題目順序或答案判斷。
- 不重畫後續步驟的 Canvas、圖表或動畫。
- 不改變手機 2×2 頁籤結構、主要版面尺寸或既有導覽網址。
- 不重新設計版權按鈕、頁尾或其他無關介面。

## 完成標準

1. 兩個例題頁左上角均穩定顯示「挑戰」。
2. 定比與倍比首頁第一步採相同手繪框配置，各自文字內容正確。
3. 四頁各自只有一個白色當前頁籤，其他頁籤均為普魯士藍底白字。
4. 手機頁籤維持 2×2，沒有溢位或文字重疊。
5. 圖表、資料點與 Canvas 教學配色未被更動。
6. 全部自動測試與 JavaScript 語法檢查通過。
